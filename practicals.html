Practical 1: Prime Number Validator

Write a program to receive an integer input from the console and establish if the number is prime. Check edge values (e.g. less than 2).

Expected Input: 17
Expected Output: 17 is a prime number
Show Code Template 
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num**0.5) + 1):
        if num % i == 0:
            return False
    return True

n = int(input("Enter number: "))
if is_prime(n):
    print(f"{n} is prime")
else:
    print(f"{n} is not prime")

Practical 2: Binary Search in Sorted List

Implement the Binary Search algorithm on a pre-sorted list of integers. Retrieve the index value of the target key, or return -1 if missing.

Expected Input: List: [2, 5, 8, 12, 16], Key: 12
Expected Output: Key found at index 3
Show Code Template 
def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

my_list = [2, 5, 8, 12, 16]
res = binary_search(my_list, 12)
print("Found at:", res)

Practical 3: Schema DDL & Constraints

Write SQL syntax statements to design a table structure called 'Student' incorporating primary key rules, unique variables, and check limits.

Constraint: student_id is PRIMARY KEY, email is UNIQUE, age >= 18.
Show SQL Template 
CREATE TABLE Student (
    student_id INT PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50),
    email VARCHAR(100) UNIQUE,
    age INT CHECK (age >= 18),
    enrollment_date DATE DEFAULT CURRENT_DATE
);
