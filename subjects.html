Subjects & Syllabi

Explore course curriculum structures, grading schedules, weekly hours, and reference materials.

CS-101

Python Programming

An introduction to algorithmic reasoning using Python. Students learn to handle conditional structures, write reusable functions, process local file streams, and model basic class structures.

Key Modules:

  • Basic syntax, operators, collections (Lists, Dictionaries)
  • Function definitions, variable scopes, parameters
  • File handling (.txt and .csv parsing)
  • Exception structures and error debugging
Weekly Hours: 4 Hours Textbook: Reema Thareja (Oxford)
CS-202

Relational DBMS & SQL

Core concepts of relational storage models. Highlights ER mapping schemas, normalization algorithms, and writing complex structured query statements in MySQL/PostgreSQL.

Key Modules:

  • Entity-Relationship mapping and Relational Model
  • SQL Schema Definition (DDL) and Data Manipulation (DML)
  • Grouping aggregates, nested joins, subqueries
  • ACID properties and Transaction logs
Weekly Hours: 5 Hours Textbook: Korth & Silberschatz
CS-301

Data Structures & Algorithms

Understanding algorithmic efficiencies and memory layout. Covers list nodes, stacks, FIFO queues, quick/merge sorting techniques, and basic binary tree operations.

Key Modules:

  • Time and space analysis (Big O notation)
  • Linked lists, Stack and Queue implementations
  • Searching and Sorting (Bubble, Quick, Merge Sort)
  • Binary Search Tree (BST) operations
Weekly Hours: 5 Hours Textbook: Narasimha Karumanchi
CS-402

Web Technologies

Foundations of modern client-side scripting. Highlights writing structured semantic HTML layout files, modular styling sheets, DOM manipulation and event handlers.

Key Modules:

  • Semantic HTML5 and accessibility layouts
  • CSS3 flexbox, grid, and customized variables
  • Vanilla Javascript DOM handling and click events
  • Asynchronous API fetches (AJAX, JSON formats)
Weekly Hours: 4 Hours Textbook: Jon Duckett (HTML/CSS/JS)
